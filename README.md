【我是誰 ? 】
------
### 我是 Richard，精通孫子兵法，擅長紙上談兵 !

喜歡物聯網跨平台整合的東東，有用  Linkit Smart 7688 搭了個小系統自娛自樂，
能開發 Java 後端、Web 前端，也能撰寫行動應用 Android 與 iOS App，

    目標是朝擁有十八般武藝的「架構師」前進，有朝一日打造自己的 iOT 系統。
